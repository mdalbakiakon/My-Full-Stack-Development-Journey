Today I am learning Color in CSS

Color can be represented in CSS following ways:
  1. Direct Color Name
  2. Hex Color Code
  3. RGB Color Code
  4. RGBA Color Code (A here means Alpha basically opacity it is 0 to 1)
  5. HSL (Hue[degree based 0 to 360 degree], Saturation[basically purity of the colour used as parcentage], Lightness[making brigter also mesuring in parcentage])
  6. HSLA (Same again A is Alpha determine the opacity of the colour from 0 to 1)
